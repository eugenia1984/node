Ejercicios

1 - Consigna: Escribe un programa de una sola línea que haga que aparezca en la pantalla un alert que diga “Hello World”.

2 - Consigna: Escribe un programa de una sola línea que escriba en la pantalla un texto que diga “Hello World” (document.write).

3 - Consigna: Escribe un programa de una sola línea que escriba en la pantalla el resultado de sumar 3 + 5.

4 - Consigna:Escribe un programa de dos líneas que pida el nombre del usuario con un prompt y escriba un texto que diga “Hola nombreUsuario”

5 - Consigna: Escribe un programa de tres líneas que pida un número, pida otro número y escriba el resultado de sumar estos dos números.
